## Advent of Code 2025 Solutions in Kotlin

My solutions for [Advent of Code challenges 2025](https://adventofcode.com/2025) in Kotlin (JVM).

Before I start I want to give a big shout-out to [Todd Ginsberg](https://github.com/tginsberg/advent-2025-kotlin) whose
solutions are very concise as well as idiomatic to read.
His repos are always very inspirational! I learned a lot from him.

Just use the main class and execute either the appropriate `solve` functions in the according `Day` classes or any `day`
function i.a.

**Enjoy!**
